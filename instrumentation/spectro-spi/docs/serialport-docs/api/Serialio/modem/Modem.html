    request

    public void request(int code)
    Request modem to perform an action. Note the wait block, this allows the caller to send multiple requests to the modem object without having to check for completion. If the previous request has not finished, the caller will block until it does. CAVEAT: If request() is called twice from the modemEvent() callback, the modem thread will deadlock
    Parameters:
    code - the request code
